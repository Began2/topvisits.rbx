# Topvisits.rbx
Topvisits.rbx is a Catweb site that tracks the most visited Catweb domains. A visits leaderboard.

## What you can contribute towards
Feel free to optimize the code, fix broken stuff, edit statistics, fix something visually broken.
Don't edit anything that would drastically change the site design wise, don't put backdoors into the code.
After a repository edit, I will update the site to match what is on github.

## Quick leaderboard criteria
- The site must be above 10k visits
- Has to be public and accessible
